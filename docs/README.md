# Documentation

Documentation for quant-tool project.

## Quick Start

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Install the package in development mode:
   ```bash
   pip install -e .
   ```

3. Check examples in the `examples/` directory.

## Modules

- **data**: Market data fetching and processing
- **strategy**: Trading strategy development
- **backtest**: Strategy backtesting engine
- **risk**: Risk management tools
- **visualization**: Chart and result visualization
